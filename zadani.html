Základy JS - opakování

1: Co je JavaScript a k jakým účelům se používá?

Doplň vhodné odpovědi a využij k tomu nečíslovaného seznamu.

Použij v textu aspoň 3 internetové odkazy na zajímavé příklady využití JS!

  • Interaktivita na webových stránkách: JavaScript umožňuje vytvářet interaktivní prvky na webových stránkách, jako jsou tlačítka, formuláře, animace, galerie obrázků, atd. Díky JavaScriptu můžete reagovat na akce uživatelů, jako je kliknutí, přetažení, stisk klávesy a mnoho dalších.

  • Validace formulářů: JavaScript se používá k validaci dat v formulářích na webových stránkách. Pomocí něj můžete zkontrolovat, zda uživatelé zadali správně své údaje nebo zda jsou povinné pole vyplněny.

  • Dynamické obsahy a AJAX: JavaScript umožňuje načítání dat a aktualizaci obsahu na webových stránkách bez potřeby celé stránky znovu načítat. Toho se dosahuje pomocí techniky nazývané AJAX (Asynchronous JavaScript and XML), což umožňuje interakci se serverem bez překreslování celé stránky.

4: Jak je možné v JavaScriptu deklarovat proměnné?

Uveďte na příkladech různé datové typy, které je možné využívat v JS.

Doplňte chybějící kód/text podle vzoru úvodních dvou řádků!

let x = 0; // s přiřazením hodnoty
let x; // bez přiřazení hodnoty
let x = null; // null = absence hodnoty
let cislo = 42; // celé číslo (typ number)
let realneCislo = 3.14; // reálné číslo (typ number)
let desetinneCislo = 6.022e23; // 6.022 x 10^23 // zápis desetinného čísla pomocí exponentu
let prazdnyText = ""; // typ string
let prestupny = false // typ boolean

10: Větvená podmínka switch a funkce

Vložte kód v JS, který do popup okna vypíše aktuální den v týdnu:

  1. Po kliknutí na odkaz "Vypiš den" se vyvolá funkce "denVTydnu".
  2. Součástí funkce "denVTydnu" je zjištění aktuálního data (pomocí objektu Date).
  3. Pomocí funkce bude zjištěno číslo dne v týdnu.
  4. S využitím příkazu switch se vypíše český název dne (pro všední dny), nebo údaj "víkend" pro sobotu a neděli.
  5. Správnost řešení ověřte výpisem výsledku do popup okna.
